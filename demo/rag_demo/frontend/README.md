# RAG UI App (frontend)

A professional, magical RAG UI for global audience engagement and LinkedIn demo.

## Features
- Stunning, modern UI/UX with MUI and Emotion
- Magical user engagement (confetti, smooth transitions)
- Prominent branding for Tusshar Lingagiri
- Augmented answers via backend RAG pipeline
- Ready for global professional showcase

## Getting Started
1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view the app.

## Backend Integration
- The `/api/rag` endpoint should be connected to your Python RAG pipeline for real augmented answers.
- Update `pages/api/rag.js` to call your backend service.
